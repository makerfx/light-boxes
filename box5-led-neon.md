# Box 5 - Non-Addressable LED Neon

## Overview
This is a very simple type of LED lighting. Simply bend the LED neon into the shape you want and plug it in!

## LED Specifications
* Addressable ("Pixels") = No
* Weather Rating = IP65
* Power Connector = "Wall wart" - 12v output
* Control = Unplug it?
* 80's Vibe = MAX!
